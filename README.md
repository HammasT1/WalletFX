# CardFlow

Animated credit card showcase built with Flutter.

## Setup

1. Run `flutter pub get`.
2. Launch with `flutter run`.
3. Tap a gradient swatch to change the card theme.
4. Drag the card to tilt it, or tap the card to flip it.
